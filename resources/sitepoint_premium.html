SitePoint Premium

SitePoint Premium is a membership resource for textbooks, video-based web courses, screencasts and other materials that SitePoint has made arrangements to share via this service. SitePoint.com is an Australian company that produces many resources for beginner through expert web developers.

Buy The Code

You must purchase an access code through the BYU-Idaho bookstore (ISBN: 281000004207B) for $11.00. Then use the access code to sign-up for the SitePoint Premium service at https://www.sitepoint.com/premium/sign-up.

Use It

Starting in week 1 you will have readings assignments, video courses and screen casts to work with as part of the course. In addition, there are many other resources for you to explore and use.
