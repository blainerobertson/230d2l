Configure eqFTP in Brackets

During week 1 you installed the Brackets text editor and a number of recommended extensions, one of which was eqFTP. This extension allows you to move the files on your computer to the remote server so those become available on the Internet. This activity is to configure this tool so that it will operate as you need it to.

Video Instructions

I have created two videos, the first demonstrates the installation and primary configuration of the tool. The second video takes you through the process of creating the actual connection to move the files to the remote location. Watch and follow along with the video, substituting your own information as you set up your tool.
