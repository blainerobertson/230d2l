Alterations Have Been Made

In the videos you will see that the presenter is using jQuery version 1.x and in lessons 3, 4 and 5 the jQuery is being used from a CDN. I have slightly altered his code in two ways:

  1. I have included a "js" folder in all lesson folders that contains the downloaded jQuery 3.x file and the cdn.txt file from the jQuery: Adding it to a web page activity.
  2. I have replaced the call to the CDN version of jQuery 1.x in all web pages with a reference to the local version of jQuery 3.x.

All other code seen in the videos remains the same and should work the same.

A Final Comment

In the videos two script elements are used: 1) the script element including the jQuery library into the page and 2) the script where the jQuery code is being written. Both are in the head element of the web page. For demonstration purposes this makes the presentation easier.

However, to conform to the web development model the jQuery code should be written in an external javascript file, then referenced using a script element's "src" attribute to be made available to the html web page as shown below.

      <footer>
           <h5>© Alcove Digital Systems | <a href="#">Terms of Use</a></h5>
      </footer>
     </div><!-- Close wrapper -->
     <script src="js/jquery-3.0.0.min.js"></script>
     <script src="js/jqscript.js"></script>
</body>
</html>

In order to allow the html to load first, before waiting for the javascript code to load, both script elements: 1) to load the jQuery library (this must be first), and 2) to load our jQuery code would be placed near the bottom of the page as demonstrated in the jQuery: Adding it to a web page activity and modeled above.
