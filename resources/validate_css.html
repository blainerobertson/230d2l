Validate CSS

Why?

Validating your CSS is important to make sure your code is written correctly and to current standards (CSS Level 3). It also increases the likelihood that your code will work correctly across all major browsers and devices today and into the future.

Where?

The primary CSS validator is run by the World Wide Web Consortium (W3.org), the same folks to create and maintain the standards for most web technologies. The actual validator can be found at http://jigsaw.w3.org/css-validator/.

How

You could use the W3 validator directly, but it slows things down. Instead I recommend using the "Validate CSS" option, in the "Tools" menu of the Web Developer tool (you should have installed it earlier in the course).
