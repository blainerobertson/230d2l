Mobile Compare and Contrast Assignment

Note: You should have completed the 24 hour mobile browsing experience prior to undertaking this assessment and completed the assigned readings for this week.

Instructions

You were asked to keep track of web sites that did a good job and those that didn't do so well as you spent your time browsing the web on your mobile device. Now I want you to start explaining the good and the bad.

Requirements

  1. Develop a web page that you will post to your web site in the homework directory. You may use this template file if you wish.
  2. There should be two sections — one for the good and one for the bad.
  3. Pick a web site that you believe provided a good visitor experience and list it in the good section.
  4. Pick a web site that you believe provided a poor visitor experience and list it in the bad section.
  5. In each section, list the site by name and provide the URL to the site.
  6. For each site explain how the site did or did not do the following:
    1. Allow you to find the content you were looking for
    2. Allow you to read the text, view the images or media associated with the text
    3. How the navigation helped you move through the site
    4. How the colors and layout of the content enhanced or distracted the content you were consuming
    5. Offer a personal statement of likes or dislikes for the site and explain why
  7. For the good site provide a screenshot of a page you think did a great job.
  8. For the bad site provide a screenshot of a page that failed.

Submitting Your Work

  1. Save the file as a web page into your web site in a folder specifically for homework (you should already have this directory (folder). If you don't, create it - remember the naming rules).
  2. Upload the file and the folder to your production web site.
  3. Add a link to the file in the temporary home page of your web site.
  4. Upload the changed temporary home page to your web site.
  5. Open your home page in a browser and make sure the link to the comparison file works.
    1. If the link doesn't work, do some troubleshooting or get help until it does.

Peer Review

After you post the page to your website, post a link to your page to the collaboration board in this week's module.

In addition to posting your link, you need to view, review and give feedback to those in your learning group.

Assignment Grade Breakdown (5 points total - Objective 3)

  • Link posted to collaboration board - 2 points
  • Responded to your group members - 3 points
