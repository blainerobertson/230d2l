Course Project Final Assessment

When

The course project web site will be assessed no earlier than 11:30pm on the last published day of classes for the semester.

How

The assessment will use the same grade sheet as was used for the peer review.

Publication

Scores will be published to the grade book only when all course projects have been assessed for the entire section.

Warning!

Carefully review each item in the grading matrix below, these are the criteria used to grade your project. Remember that if you failed to participate in the Peer Review your course project will be penalized 50%.
