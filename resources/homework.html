Homework Page

Up to this point in the course the temporary home page has held all links to assessments. With the course project taking over your hosting account as the primary content of the site, the links to assessments must be moved.

Create (if needed) a Homework Folder

  1. If you do not already have a folder where all homework is stored, create one.
  2. Be sure to follow the web naming rules.
  3. Move all files and folders containing previous homework assessments to this new folder.

Create a Homework Page

  1. Using the same page template and styling that you are using for your course project, create a new web page.
  2. It should be stored in the homework folder and have an appropriate name, following the web naming rules.
  3. It should have a meaningful, content-related title.
  4. The links that previously existed in the temporary home page should now be placed into this new page.

Link to the Homework Page

In the footer of your course project home page, located at the root of your site, create a link to the homework page.

Test, Test, Test

  1. Make sure the link from the home page to the homework page works.
  2. Make sure the links from the homework page to the assessments work.
  3. Make sure the assessments continue to work (adjust as needed until they do!)

Note: You are encouraged to collaborate and work with others while building your project site. However, the responsibility for the site is entirely yours.
