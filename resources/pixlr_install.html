Install Pixlr Extension to Brackets

Pixlr is a popular and free online image editing application - https://pixlr.com. It is also available as an extension for Brackets.

Follow the directions provided in the video and install the Pixlr extension for Brackets. It will allow you to edit images for this unit and in the future without leaving Brackets.

PhotoShop

The market standard for image work is Adobe's PhotoShop. However, it is also expensive. If you have access to PhotoShop and want to use it, please do so. If you don't and don't want to spend the money to get that access, Pixlr will allow you to perform most common image manipulation tasks.

Brackets Pixlr Extension Video

Transcript from the Video

Pixlr is a popular online image editing tool. It is also available in Brackets as an extension. Let's install it and do a quick introduction to using it.

Open the Brackets Extension Manager and make sure you are in the "Available" tab.

In the search bar, search for "pixlr".

Click the "Install" button.

When the installation is complete, close the manager window.

Restart Brackets.

Here is where things get fuzzy. The default Pixlr icon that would normally display in Brackets is broken. Instead of seeing an icon, you see a blank. The button is still there and works, just no icon.

Click the button and Pixlr opens, if you have Flash Player on your machine.

If you don't have Flash, you can download it from Adobe using the URL shown - https://get.adobe.com/flashplayer/

Once Pixlr opens, click the desired option from the menu shown on the screen.

In our case, we will open a file from our computer.

In this short video we will crop and resize the image.

With the image open, the tool bar typically is displayed on the left and the tools are fairly common to image editing programs.

Click the crop tool and select the portion of the image to be retained.

Once selected, grid lines appear helping you judge the "rule of thirds" to position the crop for appropriate composition. The grid can be manipulated as needed.

When satisfied, press your "Enter" or "Return" key. The image is now cropped.

Next, go to the image menu and select "Image size...". Here you can change the width or height (in pixels) and by default the ratio is preserved.

When done, click "OK".

Finally, to save the new version of the image, select "File" > "Save" from the Pixlr window.

Change the name, select the file format, and if the format supports compression, change the Quality using either the slider or type a number between 1 and 100.

You will see the file size shown in the bottom right corner, above the "OK" button.

When satisfied, click "OK".

A system dialog box will open asking for the destination and giving you the option to further edit the file name.

Warning! If you leave the default name and location, you will overwrite the original file.

If you want to preserve the original file, then change the name of the new file or change the location folder.

In this video, the original files are in a folder named "raw-images", I will store my revised image to a new location, named "images".

I will also add the width, in pixels, to the file name to easily know its size. In this case, the image is 500 pixels. So the name will be bridge_500.jpg.

Once named and the correct location chosen, click "Save".

When done, close the Pixlr window by clicking the "X" button in the top right corner of its window.
