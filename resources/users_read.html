How Users Read

Contrary to what we would like to believe, site visitors seldom read our content. They scan it. To better understand this critical concept and how to make it easier for users, read the article via the link below from the Nielsen / Norman group.

How Users Read...

Tip

The three techniques mentioned in the article, while sounding easy, are sometimes hard to achieve:

  • Use objective language
  • Create scannable text
  • Reduce word count

Reducing the word count can be achieved by using the "texting" mentality. If you have a typical texting app, you are limited to 140 - 160 characters which forces you to be concise. Try the same principle with text on your web page.
