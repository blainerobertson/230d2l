Responsive Media Assessment

Upload The Finished Files

With all of the activities completed: Creating Images, Optimizing Images, Implementing Images and Implementing HTML5 Video, you are to upload the entire responsive_media folder, with all of the images and videos and the css file and web page to your own web site.

Create A Link

Create a link to the index.html file within the responsive_media folder from your temporary home page.

Test Functionality

Test that the link and all of the responsive images and video work in different browsers and on different devices.

Grading Matrix

Objective 1 (10 points)

  • HTML is valid (10 points)

Objective 2 (10 points)

  • CSS is valid (10 points)

Objective 3 (25 points)

  • Web naming rules have been followed for all folders and files [web, css, images and movies] - (3 points, minus 1 for each error)
  • Images are responsive to different device sizes and/or pixel densities - (10 points)
  • Video plays in all browsers and on all devices - (5 points)
  • Total page weight is under 400k on all device sizes (5 points, all or nothing)
  • No horizontal scrolling or page zooming is required on any screen size - (2 points)
