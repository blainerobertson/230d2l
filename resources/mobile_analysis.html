Mobile Visitor Analysis and Persona

Note: You should have completed your 24 hour mobile browsing experience prior to undertaking this assessment and completed the assigned readings for this week.

Instructions

This assessment asks you to do a visitor analysis for a particular web site that you visited during your mobile browsing exercise (the site you thought did the best job is recommended). In addition to the visitor analysis you will develop a Persona representing a typical site visitor. You should use the Visitor Consideration file to guide you in the Visitor Analysis section and the Persona reading to guide you as you develop the Persona for your chosen web site.

Why are you doing this? The intent is to get you thinking critically about who we build web sites for?  The people who visit the site, not ourselves! By doing this, the hope is that when it comes time for you to build your own site, you will keep in mind who you are building it for and how you can best meet their needs.

Requirements

  1. Develop a web page that you will post to your web site. You may use this template if you wish.
  2. Pick a web site that you believe provided a good visitor experience.
  3. Clearly state the name of the site and provide the URL to the site.
  4. There should be two sections: Visitor Analysis and Persona (clearly label each section).
  5. For the Visitor Analysis section do the following:
    1. Address each item in the Visitor Consideration document for your chosen site.
    2. Provide adequate detail to demonstrate that you have carefully thought about how the site is considering and meeting the needs of the site audience and visitors.
    3. Include screenshots from the page you think did a great job and explain why.
  6. For the Persona section, include the following:
    1. All of the items represented in the Example Persona found in the Persona reading (including a picture).
    2. For the quote, provide a task or question that the persona would try to accomplish or ask while visiting the web site.

Submission Instructions

  1. Save the web page your web site in a folder specifically for homework (you may need to create this folder if it doesn't yet exist - remember the naming rules).
  2. Upload the file and the folder to your production web site.
  3. Add a link to the analysis file in the temporary home page of your web site.
  4. Upload the changed temporary home page to your web site.
  5. Open your home page in a browser and make sure the link to the analysis file works.
    1. If the link doesn't work, do some troubleshooting or get help until it does.

Submitting for Peer Review

After you post the web page to your website, post a link to the file to this week's collaboration board.

In addition to posting your link, you need to review and give feedback on the analysis page of the people in your learning group.

Assignment Grade Breakdown (5 points total) - Objective 6

  • Link posted to collaboration board - 2 points
  • Responded to all group members - 3 points
