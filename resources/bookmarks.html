Bookmarks Exercise

There are a number of useful and commonly used web resources for this course and beyond. The list of web resources can be viewed using the link below and then bookmarked within your browser for easy access at any time.

Frontend Development Resources

Expectations

It is expected that you will:

  1. Visit the resource page via the link provided above then bookmark the page in your browser.
  2. Once bookmarked you should install the browser extensions listed (Chrome extensions are noted with "Chrome Extension" while add-ons for FireFox are noted with "FireFox Add-on" to Chrome or FireFox (or both) by clicking the link in the resource list.
  3. When clicked, a new tab will open with the extension or add-on being shown with a button to install the tool to the browser. You should install all tools for the specific browser (5 add-ons for FireFox, 4 extensions for Chrome).

Adding a Bookmark

If you are unfamiliar with bookmarking a page, directions for various desktop browsers are shown below:

FireFox

  1. With FireFox open, click Bookmarks > Bookmark this page
  2. Select a folder or location in the dialog box
  3. When finished, click "Done"
  4. To revisit the page, return to the bookmarks menu > Recently Bookmarked > Web Frontend Development
  5. The Web Frontend Development list will be loaded.

Chrome

  1. With Chrome open, click Bookmarks > Bookmark this page (or click the bookmark star in the address bar).
  2. Select a folder or location in the dialog box
  3. When finished, click "Done"
  4. To revisit the page, return to the bookmarks menu > Web Frontend Development (or look in a folder if stored in one).
  5. The Web Frontend Development list will be loaded.

Safari

  1. With Safari open, click Bookmarks > Add Bookmark.
  2. Select a folder or location in the dialog box
  3. When finished, click "Add"
  4. To revisit the page, return to the bookmarks menu > Web Frontend Development (or look in a folder if stored in one).
  5. The Web Frontend Development list will be loaded.

Internet Explorer

  1. With IE open, click the Favorites button (Star icon) in the top right corner
  2. Click the "Add to Favorites" button
  3. Select a folder or location in the dialog box
  4. When finished, click "Add"
  5. To revisit the page, click the Favorites button (Star icon) in the top right corner, click on Web Frontend Development (or look in a folder if stored in one).
  6. The Web Frontend Development list will be loaded.

Opera

  1. With Opera open, click Bookmarks > Add to Bookmarks (or click the heart icon in the address bar).
  2. The page is automatically added to the bookmark list within Opera.
  3. To revisit the page, return to the bookmarks menu > Web Frontend Development.
  4. The Web Frontend Development list will be loaded.
