Upload and Test

With the page complete it is ready for its initial public appearance.

  1. On the left, in the Project panel, you should see your index.html file.
  2. Right-click it and select “Upload”.
  3. If the eqFTP tool is configured right, several things should happen:
    1. The eqFTP icon (on the right) should show a circular arrow during upload and a green checkmark when done.
    2. Green dialog boxes may apprear in the bottom right corner of the Brackets window showing the server connect and disconnect messages.
  4. You should open a browser window and go to your remote site. The temporary home page should be visible.
  5. If everything works “Hooray!!”
  6. If something fails, get help!

Use FileZilla

If the eqFTP tool doesn't work, then try using FileZilla to upload the file to the remote server.

This exercise is complete only when you go to your own domain and see the temporary home page. Get help if needed but keep at it until everything works as it should.
