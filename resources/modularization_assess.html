Modularize the Weather Pages

This week you learned about modularization of basic, common code structures into the web pages. In this assessment you will implement common code modularization into both (Franklin and Greenville) weather pages.

Implement

When either of the weather pages is visited the common content (header, nav, footer) should be identical. In addition, the pages themselves must change from using the ".html or .htm" extension to ".php". Make sure the implementation of modularization on these two pages has occurred.

Start Using XAMPP

In order to test php web pages on your local machine you will have to use XAMPP. Refer to the Testing PHP Locally document for directions.

Keep In Mind

This assessment is a trial run for your site project assessment that will occur at semester's end.

Grading Matrix
