Style the Franklin Weather Page

Introduction

This week you learned about the basics of CSS. Now it is time to put some of what you learned into action. Remember the old adage, "Practice makes perfect"!

Mobile Focus

Don't forget that we are ignoring all screens other than mobile ones. Your complete focus is to make sure the content of the Franklin weather page can be easily consumed on a mobile screen and looks good (remember the design principles of Proximity, Alignment, Repetition and Contrast, along with Typography?).

External Style Sheet

All of your CSS must be in an external stylesheet and hooked back to the web page using the <link> element (just like in the materials for this week).

Valid CSS

Be sure to validate your CSS to make sure it is written correctly.

Get a Second Opinion

If you are like me and don't really consider yourself a designer (and even if you do) then it is worth it to get a second opinion. Ask a classmate to look at the Franklin page, in a mobile environment, after your CSS is in place. Listen to what they say. Make changes as needed.
