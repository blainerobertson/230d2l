Teaching Presentation

What

During the course each student is expected to create a 7-minute presentation on a topic from the supplied topics list.

How

This presentation can be created using presentation software like PowerPoint or Keynote, or using a web-based program like Prezi, Haiku Deck, Google Slides, or something similar.

Where

The presentation will be given in class on the scheduled date. You will also create a semantic and valid web page in your own site with the same content on the page (not the same presentation). The page should have an appropriate heading with the title of the presentation topic. The presentation page should be linked to from your temporary home page.

When

The presentation will be given in class on the date scheduled.

In addition, the web page must be posted by the end of week 6 or on the date of your presentation (whichever is later).

Review

Review the Teaching Presentation Grading Matrix and the Teaching Presentation Topics List.

Sign Up for Your Presentation

After reviewing the above documents sign-up for a topic in the Teaching Presentation Sign-up spreadsheet.
