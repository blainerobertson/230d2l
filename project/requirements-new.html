Web Project Requirements

The course project is the major project of the semester and is representative of all topics and objectives of the course. The use of code taken from textbooks, video courses and other outside resources is not allowed (although the ideas and concepts from these resources is encouraged). All work must be your own, but you are encouraged to work with your peers.

The project must include all of the following:

Objective 1

  • All web pages (including the planning and homework pages) must use proper semantic markup and be valid HTML5.
  • All aspects of the site must be fully operational.
  • Folders and files (all types) must follow the web naming rules.
  • All pages (including the planning and homework pages) will have meaningful, content specific titles.

Objective 2

  • All web pages (including the planning and homework pages) must be styled using valid, external CSS3 stylesheets.
  • All web pages (including the planning and homework pages) must be responsive and adapt to small, medium and large screen sizes.
  • Media queries are appropriately used to change content layouts for large, medium and small screens. These are used in conjunction with the viewport meta element.
  • No horizontal scrolling and zooming will be needed to view content (This does not apply for content that scrolls within a container in order to display a sequential series of values, but must NOT apply to the entire page).

Objective 3

  • Usability enhancing techniques must be implemented throughout the site.
  • All web pages (including the planning and homework pages) must have a page weight of 400k or less.
  • Site images must be optimized and delivered using responsive media techniques appropriate to screen size and/or density and operating environment.
  • At least 1 video of your own creation and representative of your site content must be delivered using appropriate techniques.
  • The design principals of Proximity, Alignment, Repetition, Contrast and Typography must be recognized and used in the visual styling of the site.
  • The visual appearance of all pages (including the planning and homework pages) in the site must be uniform and consistent.

Objective 4

  • A planning document describing the site purpose, target audience, an appropriate persona and wireframe sketches for three screen sizes (small, medium and large) must be provided. The planning document must also indicate the following: 1) which pages use JSON data and 2) which page contains the content video. A link to the planning document must be found in the footer of all site pages.
  • A link to the homework document must be found in the footer of all site pages.
  • Modularization is used in all web pages to provide common content (header, navigation and footer) throughout the site.
  • Navigation must be fully operational and adapt to screen size and all links must be operational.

Objective 5

  • JSON data (of your own creation) must be used as the primary content on at least 2 pages in the site.
  • JSON data must be inserted using JavaScript and/or JQuery. Other uses of JavaScript and/or jQuery are encouraged to enhance the user experience.

Objective 6

  • A minimum of 5 content pages (excluding the planning and homework pages) must be present in the site.
  • The project must be finished and ready for assessment on time.
  • The project must be the main content of the web site and present by the due date.

Project Specifications

At or near week 12 you will be given a specifications document for the project. From the specifications document you will built your own planning document and the finished web site as described above. Carefully review the specifications document when it is made available and ask questions as you have them.

Project Submission

The finished project is due at 12:00pm (noon) on the last class day of the semester. To submit, post all pages for this project to your website, as the main content of your site. All previous assessments will be found in the footer of all pages in the finished project site found via a "homework" link.

Note: This is a web course intended to help you learn how to construct web sites using HTML5, CSS and JavaScript. Outside templates are not allowed (except as introduced in the course). Pages built from site builder software or drag-n-drop tools are NOT allowed.
