Acme Project Grading Matrix

Course Objectives

All six course objectives will be assessed in the Acme project. Each objective is valued at 30 points. The specifications listed below apply to all project web pages.

Objective 1

10 points - All web pages use semantic elements for page and content structures (remember that div and span elements should be used only as a last resort, when no other semantic element is available). 3 points will be deducted for each missing semantic element.

10 points - All web pages are valid HTML5. 2 points will be deducted for each validation error found by the HTML5 validator.

5 points - All folders and files (web pages, images, videos, javascript, json, css and any other site file) follow the web naming rules. 2 points will be deducted for each naming rule error found in the site.

5 points - All web pages have meaningful, content-related titles. 5 points will be deducted if any page is missing or lacks a specific content-related title.

Objective 2

10 points - All pages are valid CSS3. 2 points will be deducted for each validation error found by the CSS validator.

5 points - Page and content layouts adapt as screen size changes and no horizontal scrolling or zooming is required to view content (including images), on any screen size. All or nothing score, if content does not adapt or requires horizontal scrolling or zooming to view the content, all 5 points will be deducted.

5 points - All CSS must be written and applied to the site using external CSS pages. 5 points will be deducted if any internal or inline css code is found.

10 points - Examine the CSS.

  1. Does the style sheet begin with a "mobile first" approach? (Subtract 3 points if no)
  2. Is at least one media query present AND used to change content layouts for screen sizes larger than mobile? (Subtract 3 points if no)
  3. Is Flexbox used as the primary means of implementing responsive layout? (Subract 4 points if no)

Objective 3

10 points - Usability principles from the Usability Checklist (focusing on the five "low hanging fruit" items) must be applied to content in the Acme site. 3 points will be deducted for each missing usability concept up to 10 points.

5 points - The uncached weight of Acme pages must be equal to or less than 400k. 5 points will be deducted if the weight is greater than 400k.

10 points - The PARC and typography design principles must be found applied to the style of the Acme project site. 3 points will be deducted for each missing design principal up to 10 points maximum.

5 points - The visual appearance must be uniform and consistent across the Acme project site. 5 points will be deducted if one of the project site pages does not follow the visual appearance of other pages in the site.

Objective 4

10 points - The template file contains all web page components (including navigation) but no content. It is valid HTML5 and CSS3. All or nothing score, 10 points if yes, 0 if no.

10 points - jQuery and JSON data is used to provide the navigation links for all Acme site pages. Subtract 10 points if not delivered from the JSON file using jQuery/JavaScript.

10 points - Navigation must be fully operational and adapt to screen size. Subtract 5 points for each navigation item that does not work. Subtract 5 points if navigation fails to adapt as screen sizes change.

Objective 5

30 points - JSON data is the primary content of all four content pages. The JSON object being used for the content page must be displayed in the javascript console of the developer tools within the browser.

  1. Use the navigation bar to navigate to the content page. Does the content appear in the browser window?
  2. Inspect the page, click the console tab in the developer tools, the JSON object should be visible. Expand it and examine the object values. Is the object visible? Is it JSON?

Deduct 10 points per content page that does not use the JSON data for it's content or if the content is not operational or missing from the console.log window.

Objective 6

10 points - The Acme project must be accessible using a link on your home page. The project must have a minimum of a home and four content pages. 10 points will be deducted if less than five pages are found in the Acme site.

10 points - The content of the Acme site is free of spelling and grammar errors. 2 points will be deducted for each error found.

10 points - The Acme project demonstration video URL is present and:

  • Fits the 5 minute maximum time limit
  • Demonstrates the navigation bar being clicked and the content being delivered.
  • Demonstrates the responsiveness of the home and content pages.
  • Demonstrates of the validity of the HTML5 and CSS3 code for the home and content pages.
