This week we continue using JavaScript, jQuery and add some PHP. Our specific goal this week is to make a web page "Bulletproof". If JavaScript is disabled, jQuery and JavaScript don't work and that means our content can be dead or gone. This is not a good thing. We want our site to work regardless, that is the point of this week.
