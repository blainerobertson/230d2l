Event Handlers

In order to make a website interactive using jQuery (and JavaScript) you must understand events. In this activity you will review events and event handlers.

  • Review jQuery.com's Event Object Documentation (in particular, review event.stopPropagation() and event.preventDefault()).
  • Review jQuery.com's Learning Center for event handlers
  • Review jQuery.com's Learning Center for event delegation
  • Review jQuery.com's Learning Center for triggering events
  • You may also want to review the videos in lesson 5 of the Introduction to jQuery video course (SitePoint Premium)
    • Lesson 5.1 - Introduction to Event Listeners (~ 7:42 minutes)
    • Lesson 5.2 - Mouse Events (~21:14 minutes)
    • Lesson 5.3 - Keyboard Events (~10:46 minutes)
    • Lesson 5.4 - Form Events (~6:31 minutes)
    • Lesson 5.5 - Best Practices (~10:26 minutes)
The purpose of this activity is understand how to write and use jQuery to handle (including stopping a default event behavior and stopping an event from "bubbling up" the DOM) an event to perform an action that you want to occur. You will use event handling in this week's assessment.
